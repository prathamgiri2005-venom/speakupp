# Speakupp - Confidence Coach Landing Page PRD

## Original Problem Statement
Create a modern, clean, high-converting landing page for a confidence coach named Sandhya.
- Company Name: Speakupp
- Goal: Sell Zoom-based confidence-building classes and encourage users to sign up
- Style: Minimal, elegant, feminine but professional with soft colors (beige, white, light gold, pastel tones)
- Time Zone: IST (Indian Standard Time)
- Currency: INR (₹)

## User Personas
1. **Working Professionals** - Looking to improve presentation and communication skills
2. **Entrepreneurs** - Need confidence for pitching and networking
3. **Students/Fresh Graduates** - Want to overcome public speaking anxiety

## Core Requirements
- Hero section with Sandhya's actual photos
- About section with bio and experience stats
- Benefits section highlighting value propositions
- Video placeholder section
- Pricing section with 3 plans in INR
- Schedule section with IST timings
- Testimonials section
- Call-to-action section
- Footer with contact and social links

## Tech Stack
- **Frontend**: React 18, Tailwind CSS, Framer Motion
- **Backend**: FastAPI (minimal API endpoints)
- **Styling**: Custom theme (beige, gold, cream palette)
- **Fonts**: Playfair Display (headings), Outfit (body)
- **Animations**: Lenis smooth scrolling, Framer Motion
